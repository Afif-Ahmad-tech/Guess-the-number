# 🎮 Number Guessing Game

A fun Python number guessing game with hints and emojis!

## ✨ Features

* 🎯 Random number between 1 and 100
* 🔢 7 attempts to guess correctly
* 💡 Smart hint system

  * ❄️ Very Far Away
  * 🙂 Getting Closer
  * 🔥 Nearby
  * 🚨 Extremely Close
  * ⚡ Just Missed
* 🏆 Win screen
* 💀 Game Over screen
* 🔄 Replay option

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/number-guessing-game.git
cd number-guessing-game
```

Run the game:

```bash
python game.py
```

## 🎮 How to Play

1. Run the game.
2. Guess a number between 1 and 100.
3. Use the hints to get closer.
4. Win before your 7 attempts run out!

```text
🎮 NUMBER GUESSING GAME 🎮

📉 Too Low! 🔥 Nearby
```
